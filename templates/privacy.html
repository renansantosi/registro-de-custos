Política de Privacidade

Última atualização: março de 2026

Seus dados são privados e não são compartilhados com terceiros. Cada usuário acessa apenas suas próprias informações.

1. Dados Coletados

Para o funcionamento do sistema, coletamos os seguintes dados:

  • Cadastro: nome completo e endereço de e-mail
  • Autenticação: senha armazenada de forma criptografada (hash bcrypt)
  • Conteúdo: dados de imóveis, lançamentos de custos e documentos enviados pelo usuário
  • Logs de acesso: registros de requisições para fins de monitoramento e segurança

2. Como Usamos os Dados

Os dados coletados são utilizados exclusivamente para:

  • Autenticar e identificar o usuário no sistema
  • Exibir e gerenciar os imóveis e custos cadastrados pelo próprio usuário
  • Enviar e-mails de redefinição de senha quando solicitado
  • Gerar relatórios exportados em Excel

3. Isolamento de Dados por Usuário

Cada conta de usuário é completamente isolada. Um usuário não tem acesso aos dados de outro usuário. Todas as consultas ao banco de dados são filtradas pelo identificador único do usuário autenticado.

4. Armazenamento e Segurança

  • Os dados são armazenados em banco de dados PostgreSQL hospedado na plataforma Render.com
  • As senhas são armazenadas como hash criptográfico e nunca em texto simples
  • As comunicações entre o navegador e o servidor utilizam HTTPS
  • Tokens de redefinição de senha expiram em 1 hora

5. Compartilhamento de Dados

Não vendemos, alugamos nem compartilhamos seus dados pessoais com terceiros, exceto quando exigido por lei ou ordem judicial.

6. Retenção de Dados

Seus dados são retidos enquanto sua conta estiver ativa. Para solicitar a exclusão de sua conta e todos os dados associados, entre em contato pelo e-mail abaixo.

7. Cookies e Sessão

O sistema utiliza cookies de sessão para manter o usuário autenticado. Esses cookies são essenciais para o funcionamento do sistema e não são utilizados para rastreamento ou publicidade.

8. Seus Direitos

Você tem direito a:

  • Acessar os dados que temos sobre você
  • Corrigir informações incorretas
  • Solicitar a exclusão de sua conta e dados
  • Exportar seus dados (disponível via exportação Excel)
